# Taskmeister

Simple task management modelled after [t](http://stevelosh.com/projects/t/).

## Installation

Add this line to your application's Gemfile:

    gem 'taskmeister'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install taskmeister
